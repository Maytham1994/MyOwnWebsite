Plants vs. Zombies using Java

In this project, a Java program was designed to simulate a plants vs zombies game.
The game is a popular game on phones, tablets and computers.

This project was done through 5 assignments in an object oriented programming course. Using Java, different classes were created to simulate different parts of the game. The final part of the project was to create a GUI to represent the game. The picture on the right shows the game when it first runs.

The Classes Are:

  • Garden (Main class). // Represents the game space.
  • GardenItem (Abstract). // Represents the different items in the garden.
  • Pea (extends GardenItem). // Represents the peas shot by the peashooter.
  • Zombie (extends GardenItem). // Represents the zombies in the game.
  • Plant (Abstract, extends GardenItem). // Represents the different plants in the game.
  • PeaShooter (extends Plant). // Represents the peashooters in the game.
  • Sunflower (extends Plant). // Represents the sunflowers in the game.
  • GraphicalGarden (extends Garden). // Contains the GUI implementation of the Garden.
  • ButtonListener (implements ActionListener). // Used to identify when a button in the garden is pressed.

The letters in the GUI represent (P: PeaShooter, S: Sunflower, o: Pea, Z:Zombie)
The user wins, when the number of zombies that have entered the garden and were killed reaches 10. If a zombie reaches the end of the garden (goes past the final location (left)), the player loses. A message indicating the player lost gets shown.
