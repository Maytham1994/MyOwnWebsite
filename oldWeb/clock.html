Raspberry Pi Clock

In this project, a clock was created using a Raspberry Pi and a 4 digit 7 segment display.
First the 4 digit 7 segment display was connected as seen in the two pictures below.
The connection uses the I2C technology to connect the display to the Raspberry Pi.
Then using the Raspberry Pi, the Python script was created.
The Python script was then set to make it run every time the Raspberry Pi boots up.
The LED seen in the pictures is turned ON when the time is between 10PM and 6AM.
This project was inspired by Dr. Monk's DIY Electronics Blog.

The Equipment & Software Used Are:

  • Raspberry Pi
  • Cobbler
  • 4 digit 7 segment display and Backpack
  • Level Converter
  • Breadboard
  • Jumper Wires
