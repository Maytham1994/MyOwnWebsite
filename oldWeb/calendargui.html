MCalendar using Java

In this project, the calendar designed previously with C++ was implemented using a Java GUI.
The same algorithms were used, the only new part was to get the GUI to work with the algorithms to produce a nice calendar.

The picture on the right shows the beta version of the MCalendar:

  • The initial month displayed is the current month.
  • The user can specify a month by clicking on the months buttons.
  • The user can choose a year, by inputting the year using the number at the top. (Year >=1900).
  • The buttons to input year are configured for 4 digit years. Therefore the maximum year is 9999.
  • The user can choose to see the next or the previous months.
