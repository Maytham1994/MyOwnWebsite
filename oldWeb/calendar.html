Calendar using C++

In this project, a C++ program was designed to produce monthly calendars for any month and year requested.
As long as the year is 1900 or greater
In-order to get this program to work, different functions; each with a specific purpose were created.
...

Program with already inputted requests

The Functions Are:

  • bool isLeap (int year); // Finds if the year is leap
  • int daysInMonth (int month, int year); // Finds how many days in the month
  • int daysSinceJan1 (int day, int month, int year); // Finds how many days since January 1st
  • int daysInPreviousYears (int year); // Finds how many days in the year
  • int daysSinceStartOfTime (int day, int month, int year); // Finds the days since Janurary 1st, 1900
  • int dayOfWeek (int day, int month, int year); // Finds the day of the week
  • void createCalendar (int month, int year); // Creates the calendar
    •
The main function in the C++ program takes care of the input and calls the previous listed function in-order to generate the calendar and print it.
To request a specific month's calendar, the month and the year are entered as shown in the second picture. To stop the program the user needs to input 0 0.
